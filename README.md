# 灰锥检测系统

## 系统概述

本系统是一个基于Web的灰锥样品侧视图检测工具，用于识别灰锥是否满足等腰三角形比例要求。系统采用先进的计算机视觉算法，结合交互式Web界面，提供高精度的灰锥轮廓提取、三角形拟合和比例验证功能。

## 核心功能

- ✅ 基于Web的交互式用户界面
- ✅ 自动灰锥轮廓提取和三角形拟合
- ✅ 多种图像预处理策略，适应不同光线和背景条件
- ✅ 手动顶点选择修正功能（提高识别精度）
- ✅ 实时检测结果显示，包括比例、高度、底边等参数
- ✅ 支持顶点在不同方向的灰锥检测（手动选点模式）
- ✅ 完善的错误处理和用户反馈机制

## 技术栈

- **后端框架**：Flask
- **计算机视觉**：OpenCV
- **图像处理**：NumPy
- **前端技术**：HTML, JavaScript
- **依赖管理**：Python pip

## 使用方法

### 1. 自动检测流程

1. **上传图像**：点击"选择文件"按钮，选择要检测的灰锥图像
2. **开始检测**：点击"检测灰锥"按钮，系统将自动进行图像处理和分析
3. **查看结果**：系统将显示检测结果，包括：
   - 是否符合灰锥比例要求（高度/底边，要求范围2.5-3.3）
   - 实际比例值
   - 高度和底边尺寸
   - 是否为近似等腰三角形
4. **结果可视化**：
   - 原始图像
   - 二值化处理后的图像
   - 标记了检测轮廓和三角形的结果图像

### 2. 手动修正流程

如果自动检测结果不满意，可以使用手动修正功能：

1. **点击进入手动模式**：在自动检测结果页面点击"手动修正"按钮
2. **选择顶点**：在图像上依次点击三个点：
   - 第一个点：灰锥的顶点（尖点）
   - 第二个点：底边左侧端点
   - 第三个点：底边右侧端点
3. **确认选择**：点击"确认检测"按钮，系统将根据手动选择的顶点重新计算
4. **查看修正结果**：系统将显示手动修正后的检测结果

## 技术实现细节

### 图像处理流程

1. **图像加载**：支持多种图像格式，自动处理图像解码
2. **预处理**：灰度化 → 高斯模糊 → 自适应阈值二值化
3. **轮廓提取**：使用OpenCV的findContours函数提取图像轮廓
4. **三角形拟合**：
   - 使用approxPolyDP函数进行多边形近似
   - 尝试不同的epsilon值以获得最佳拟合
   - 对于复杂轮廓使用凸包算法辅助
5. **属性计算**：
   - 识别三角形顶点和底边
   - 计算高度、底边长度和比例
   - 判断是否为等腰三角形

### 比例验证标准

- **比例要求**：灰锥高度与底边比例应为2.5-3.3
- **计算公式**：比例 = 高度 / 底边长度
- **判定结果**：
  - 比例在范围内：符合灰锥检测要求
  - 比例超出范围：不符合灰锥检测要求

## 注意事项

1. **图像质量要求**：
   - 灰锥在图像中应清晰可见，边缘分明
   - 建议使用简单背景，避免复杂背景干扰
   - 保持适当光照，避免强烈反光或阴影
   - 从正面水平拍摄，避免透视变形

2. **手动选点技巧**：
   - 点击时尽量准确地指向灰锥的顶点位置
   - 确保三个点形成一个合理的三角形
   - 第一个点击的点将被视为顶点，后两个点作为底边端点

3. **特殊情况处理**：
   - 对于顶点在右侧的灰锥，使用手动选点功能更准确
   - 对于复杂背景或低对比度图像，可能需要手动修正

## 常见问题解答

### 1. 为什么自动检测失败？

可能的原因：
- 图像质量较差，灰锥边缘不清晰
- 背景过于复杂，干扰轮廓提取
- 光照条件不好，对比度低
- 灰锥在图像中占比过小

解决方法：
- 使用手动选点功能
- 改善图像质量后重新检测
- 确保灰锥在图像中占据主要部分

### 2. 如何提高自动检测成功率？

- 确保图像中灰锥边缘清晰可见
- 使用均匀的照明，避免阴影
- 背景尽量简单，减少干扰元素
- 从适当距离拍摄，使灰锥占据图像的主要部分
- 避免图像过度曝光或欠曝光

### 3. 手动选择顶点时需要注意什么？

- 按照顶点 → 底边左侧点 → 底边右侧点的顺序点击
- 点击位置要准确，尽量点击灰锥的实际顶点位置
- 如果选择错误，可以重新进入手动模式重新选择
