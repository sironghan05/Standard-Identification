# 灰锥检测系统使用指南

## 系统概述

本系统用于识别灰锥样品侧视图是否满足等腰三角形比例要求。系统具有高精度检测能力、手动修正功能和完善的错误分析。目前项目使用精简版的`simplified_cone_detector.py`作为主要程序。

## 核心功能

- ✅ 自动灰锥轮廓提取和三角形拟合
- ✅ 支持中文路径图像加载
- ✅ 多种图像预处理策略，适应不同光线和背景
- ✅ 手动顶点选择修正功能（提高识别精度）
- ✅ 详细的错误分析和改进建议
- ✅ 交互式用户界面
- ✅ 支持顶点在右侧的灰锥检测（手动选点模式）

## 安装依赖

```bash
pip install -r requirements.txt
```

确保已安装以下Python库：
- opencv-python
- numpy
- matplotlib
- pillow

## 使用方法

### 1. 基本使用

运行主程序：

```bash
python simplified_cone_detector.py
```

程序会提示输入图像路径，支持中文路径，或直接按Enter使用默认图像。

### 2. 操作流程

1. **输入图像路径**：程序启动后，输入灰锥图像的路径，或直接按Enter使用默认图像

2. **自动检测**：系统会自动进行图像处理、轮廓提取和三角形拟合

3. **查看结果**：程序会显示检测结果，包括：
   - 是否符合灰锥比例要求（高度/底边，要求范围2.5-3.3）
   - 实际比例值
   - 高度和底边尺寸
   - 是否为近似等腰三角形

4. **手动修正**：如果自动检测结果不满意，系统会询问是否需要手动选择顶点：
   - 按'y'进入手动选择模式
   - 在图像上点击三个点（建议顺序：顶部/右侧顶点、左下、右下/左侧底部）
   - 选择完成后按Enter确认，或按ESC重新选择

5. **退出程序**：按Ctrl+C随时退出程序

## 手动修正功能详细说明

### 启动手动修正

当自动检测不准确时，可以使用手动修正功能：
1. 自动检测完成后，输入`y`启动手动顶点选择
2. 在图像窗口中点击灰锥的三个顶点
3. 选择完成后按`Enter`键确认，按`ESC`键重新选择

### 顶点选择技巧

1. **对于顶点在上方的灰锥**：
   - 首先点击灰锥的顶部顶点
   - 然后点击左侧底部顶点
   - 最后点击右侧底部顶点

2. **对于顶点在右侧的灰锥**：
   - 首先点击灰锥的右侧顶点
   - 然后点击左上底部顶点
   - 最后点击左下底部顶点

3. **注意事项**：
   - 点击时尽量准确地指向灰锥的顶点位置
   - 确保三个点形成一个合理的三角形
   - 第一个点击的点将被视为顶点，后两个点作为底边

## 技术参数

- **比例要求**：灰锥高度与底边比例应为2.5-3.3
- **处理方法**：自适应阈值二值化 + 轮廓提取 + 三角形拟合
- **用户交互**：支持鼠标点击选择顶点，键盘按键确认

## 注意事项

1. 为获得最佳效果，请确保灰锥在图像中清晰可见
2. 建议使用简单背景，避免复杂背景干扰
3. 保持适当光照，避免强烈反光或阴影
4. 从正面水平拍摄，避免透视变形
5. 对于特殊朝向的灰锥（如顶点在右侧），请使用手动选点功能

## 故障排除

程序会提供详细的错误分析和改进建议，请根据提示调整图像或拍摄条件。

### 常见问题解答

#### 1. 为什么比例计算结果不符合预期？

系统计算的是高度/底边比例，要求范围为2.5-3.3。如果自动检测的顶点位置不准确，建议使用手动选点功能。

#### 2. 如何提高检测成功率？

- 确保图像中灰锥边缘清晰可见
- 避免强光和阴影
- 使用均匀的照明
- 确保灰锥占据图像的主要部分
- 背景尽量简单，减少干扰

#### 3. 手动选择顶点时应该注意什么？

- 建议按照：顶点 → 底边第一个点 → 底边第二个点的顺序点击
- 点击时尽量准确地指向灰锥的顶点位置
- 对于非标准朝向的灰锥，顶点选择顺序尤为重要
