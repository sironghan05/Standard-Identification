# Standard Identification 灰锥识别系统 

## 项目简介

灰锥识别系统是一个基于计算机视觉技术的高精度灰锥自动识别工具，支持自动识别与手动修正，满足工业标准检测需求。系统已部署至 PythonAnywhere 平台，可通过 Web 界面直接使用。

**在线演示地址：** [https://sironghan03.pythonanywhere.com/](https://sironghan03.pythonanywhere.com/)

## 核心功能

- **自动灰锥识别**：通过图像预处理、轮廓提取和三角形拟合算法，自动识别灰锥形状并计算关键参数
- **手动顶点修正**：当自动检测结果不准确时，支持用户手动选择灰锥顶点进行修正
- **实时结果可视化**：显示原始图像、二值化处理结果、轮廓检测结果和最终识别结果
- **颜色自适应**：支持不同颜色灰锥（黑色、棕黄色、灰色、浅色）在各种背景上的检测
- **多策略融合**：采用多种图像预处理和轮廓提取策略，提高检测准确性
- **详细结果分析**：计算灰锥的高度、底边长度、比例等参数，并判断是否符合工业标准要求

## 技术原理

1. **图像预处理**：
   - 灰度化转换
   - 多级高斯模糊
   - 自适应阈值二值化
   - 形态学操作（开操作、闭操作、膨胀）
   - 边缘检测增强

2. **轮廓提取**：
   - 多种轮廓提取模式（外部轮廓、所有轮廓）
   - 轮廓评分系统（面积、周长、形状特征）
   - 智能筛选最可能是灰锥的轮廓

3. **三角形拟合**：
   - 多边形近似算法
   - 凸包处理
   - 特征点三角形构建
   - 最小外接矩形辅助

4. **参数计算**：
   - 三角形高度计算
   - 底边长度测量
   - 比例分析
   - 等腰三角形判断

5. **标准验证**：
   - 高度与底边比例验证（符合工业标准要求）
   - 误差范围评估
   - 详细结果判定

## 使用指南

### 自动检测流程

1. **上传图像**：点击上传区域或拖放图像文件，支持 JPG、PNG、BMP 等格式
2. **开始检测**：点击「开始检测」按钮，系统自动分析图像并识别灰锥
3. **查看结果**：系统显示检测结果，包括高度、底边长度、比例及判定结论

### 手动修正流程

1. **启动手动选点**：点击「开始手动选点」按钮，进入选点模式
2. **选择顶点**：按顺序点击灰锥的三个顶点（建议：顶部、左下、右下）
3. **确认选择**：选择完三个点后，系统自动进行计算分析
4. **查看修正结果**：系统显示基于手动选点的检测结果和判定结论

### 拍摄建议

- 在光线均匀的环境下拍摄，避免强光和阴影
- 确保灰锥占据图像中心位置，且图像清晰
- 保持背景简洁，避免与灰锥颜色相近的物体
- 确保图像中只有灰锥，或调整拍摄角度和光照

## 核心模块说明

### 1. app.py

- **功能**：Flask Web 应用主文件，处理 HTTP 请求和响应
- **主要路由**：
  - `/`：首页，展示上传界面
  - `/detect`：自动检测接口，处理图像上传和自动识别
  - `/manual-detect`：手动检测接口，处理用户手动选点修正
- **核心流程**：接收图像 → 预处理 → 轮廓提取 → 三角形拟合 → 参数计算 → 返回结果

### 2. real_photo_detector.py

- **功能**：实现核心的灰锥识别算法
- **主要函数**：
  - `load_image()`：加载图像并进行基本检查
  - `preprocess_image_color_adaptive()`：颜色自适应的图像预处理
  - `preprocess_image_adaptive()`：增强版自适应阈值方法
  - `find_largest_contour()`：增强版轮廓检测与筛选
  - `approximate_triangle()`：增强版多边形近似，优化三角形拟合
  - `calculate_triangle_properties()`：计算三角形的高和边长比例
  - `check_cone_requirements()`：检查灰锥是否满足比例要求
  - `detect_real_cone()`：多策略融合的灰锥检测主函数

## 常见问题与解决方案

### 1. 无法识别灰锥

**可能原因**：
- 光照条件不佳
- 灰锥颜色与背景相近
- 图像模糊或分辨率过低
- 灰锥未完全显示在图像中

**解决方案**：
- 调整拍摄角度和光照
- 确保灰锥占据图像中心位置
- 提高图像清晰度
- 使用手动选点功能进行修正

### 2. 识别结果不准确

**可能原因**：
- 灰锥形状不规则
- 图像中存在干扰物体
- 自动检测算法局限性

**解决方案**：
- 使用手动选点功能进行修正
- 确保图像中只有灰锥
- 调整拍摄角度，使灰锥形状更清晰

### 3. 系统响应缓慢

**可能原因**：
- 图像尺寸过大
- 服务器资源限制
- 网络连接问题

**解决方案**：
- 上传适中尺寸的图像
- 等待系统处理完成
- 检查网络连接
